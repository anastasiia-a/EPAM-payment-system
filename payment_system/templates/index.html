API Documentation

1. Получение информации о всех кошельках.
Необходимо выполнить GET запрос следующего вида:
  • https://nsokolova.samartsev.com/wallets/

    • 2. Получение информации о конкретном кошельке.
    Необходимо выполнить GET запрос следующего вида:
  • https://nsokolova.samartsev.com/wallets/{wallet_id}/

  • WALLET_ID - ID выбранного кошелька.

    • 3. Добавление нового кошелька.
    Необходимо выполнить POST запрос:
  • https://nsokolova.samartsev.com/wallets/

    • Передав в body следущую структуру:
    {
       "name" : "some_name",
       "client_firstname" : "firstname",
       "client_surname" : "surname",
    }
  • NAME - название кошелька;
  • CLIENT_FIRSTNAME - имя клиента;
  • CLIENT_SURNAME - фамилия клиента.

    • 4. Изменение информации о кошельке.
    Необходимо выполнить POST запрос:
  • https://nsokolova.samartsev.com/wallets/{wallet_id}/

    • Передав в body следущую структуру:
    {
       "name" : "new_name",
       "client_firstname" : "new_firstname",
       "client_surname" : "new_surname",
    }
  • WALLET_ID - ID выбранного кошелька.
  • NAME - новое название кошелька;
  • CLIENT_FIRSTNAME - новое имя клиента;
  • CLIENT_SURNAME - новая фамилия клиента.

    • 5. Удаление кошелька.
    Необходимо выполнить DELETE запрос следующего вида:
  • https://nsokolova.samartsev.com/wallets/{wallet_id}/

  • WALLET_ID - ID выбранного кошелька.

    • 6. Пополнение баланса кошелька.
    Необходимо выполнить POST запрос:
  • https://nsokolova.samartsev.com/wallets/{wallet_id}/deposits/

    • Передав в body следущую структуру:
    {
       "amount" : "10000"
    }
  • WALLET_ID - ID выбранного кошелька.
  • AMOUNT - сумма пополнения.

    • 7. Перевод денег с одного кошешлька на другой.
    Необходимо выполнить POST запрос:
  • https://nsokolova.samartsev.com/wallets/{wallet_sender} /withdrawals/{wallet_receiver}/

    • Передав в body следущую структуру:
    {
       "amount" : "10000"
    }
  • WALLET_SENDER - номер кошелька отправителя;
  • WALLET_RECEIVER - номер кошелька получателя;
  • AMOUNT - сумма перевода.

    • 8. Просмотр всех операций по кошельку.
    Необходимо выполнить GET запрос:
  • https://nsokolova.samartsev.com/operations/{wallet_id}/

    • В body можно передать параметр фильтрации:
    {
       "filter" : "date"
    }
    "date" - фильтрация по возрастанию даты проведения операции;
    "-date" - фильтрация по убыванию даты проведения операции.

    9. Просмотр конкретных операций по кошельку (пополнение/вывод).
    Необходимо выполнить GET запрос:
  • https://nsokolova.samartsev.com/operations/{wallet_id}/ {operation_name}/

    • В body можно передать параметр фильтрации:
    {
       "filter" : "date"
    }
  • OPERATION_NAME:
    • deposits - пополнение кошешька;
    • withdrawal - вывод денег с кошелька;
    • "date" - фильтрация по возрастанию даты проведения операции;
    "-date" - фильтрация по убыванию даты проведения операции.
